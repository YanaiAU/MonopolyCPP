Monopoly Game - QT C++ Project

This is a C++ version of Monopoly for 2-8 players, using a Qt graphical interface. Players buy properties, build houses and hotels, and make lots of money.

How to Play:

-The game board has 40 spaces, including streets, railroads, utilities, and special spaces.
-Players roll dice, move, buy properties, build houses/hotels, and collect rent.
-You can add custom squares to the board, making it dynamic.

Running the Game:
-First, you must clone the repository from my GitHub using: 
-"git clone https://github.com/YanaiAU/MonopolyCPP.git".
-Using Qt Creator makes building and running easier, run the game by executing main.cpp.
